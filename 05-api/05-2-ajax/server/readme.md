프론트 웹 어플리케이션과 연동하기 위한 서버 app (백엔드)
네트웍으로 연결되는 다른 컴퓨터에서 실행되도 상관 없다
이 컴퓨터에서 돌린다면 이 컴퓨터가 server역할도 하겠다는 의미
폴더 위치는 상관 없고 아무곳에서나 돌려도 됨 

front web과 연동하기 위한 back end web app은 다양한 언어로 개발
수업이 js여서 js node.js에서 동작하는 서버로 테스트한다 

----------서버 설치되어 있어야 함 -------

 back end web이 node.js에서 실행 
 ==>node.js 설치되어 있어야함 
 --------------------------------------
 node : js파일 실행
 >node a.js (node명령으로 실행)
npm: node package manager 
다양한 라이브러리 다운로드 및 관리 툴 
 -----------------------------
강사가 공유한 백엔드 복사 
//백엔드에서 필요한 package(라이브러리) 다운로드
server폴더로 가서 
npm install
 -------------------------------------
//백엔드 실행
node main.js
